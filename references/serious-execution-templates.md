# Serious Execution Templates

```markdown
Goal:

Success Criteria:
Unknowns:
Owner:
Advisors:

```

## Brain Decision

```markdown
Brain:
- owner:
- primary judgment:
- accepted advisor input:
- rejected advisor input:
- why:
- next decision point:
```

## Hands Contract

```markdown
Hands Contract:
- task:
- input:
- allowed write scope:
- tools:
- expected output:
- validation:
- failure codes:
- retry policy:
```

## Round Status

```markdown
Round Status:
- round:
- phase:
- completed:
- current blocker:
- validation run:
- needs user decision:
```

## Verification

```markdown
Verification:
- build:
- test:
- lint:
- manual check:
- evidence:
```

如果某项不能运行，必须写明：

- 为什么不能运行
- 是否是环境限制
- 替代验证是什么
- 剩余风险是什么
