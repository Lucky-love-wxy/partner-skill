# 05 Decisions

## 关键决策与代表性判断

### 1. Intel 从内存转向微处理器

这是 Grove 相关材料里最关键的战略原型案例。

为什么重要：

- 不是优化旧业务
- 是承认老优势已经不再可靠
- 是在组织身份层面做切换

它代表 Grove 的一个底层动作：
先承认现实变了，再重写公司真正押注的对象。

  - https://www.wired.com/1996/12/just-outta-beta-groves-new-book-looks-inside-intel

### 2. Pentium bug 危机

这个案例的重要性不在 bug 本身，而在 Grove 后来如何解释它：

- 工程师视角和公众视角之间出现错位
- 组织把一个“看起来小”的问题误判成噪音
- 结果变成品牌和信任危机

这个案例让 Grove 的框架从“运营问题”升级为“信号识别问题”。

主要来源：

- Stanford GSB on strategic inflection points

- Wired on `Only the Paranoid Survive`

### 3. Intel Inside / Pentium 品牌化与细分

外部资料显示 Grove 并不把品牌当浪漫表达，而把它当战略武器和市场接口。

这说明：

- 他不会只看技术优劣
- 他也会看命名、市场教育和品牌结构
- 但这些动作仍然服务于更大的经营控制

主要来源：

- Intel archived speech listing
  - https://www.intel.com/pressroom/archive/speeches/ag080998.htm

### 4. 个人健康议题上的系统思维

Grove 在癌症与帕金森病相关公共表达里，也延续了同一种模式：

- 不停留在同情和姿态
- 强调信息、系统、效率和可执行改进

这说明他的系统脑并不只在半导体行业里有效。

## 决策风格提炼

1. 先问现实是否已经变化，而不是先保护旧身份。
2. 先承认信号，再讨论代价。
3. 允许组织短期混乱，但不能允许长期自欺。
4. 不是所有问题都要扩大组织动作，很多时候要重写接口和控制回路。
