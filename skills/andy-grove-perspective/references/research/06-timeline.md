# 06 Timeline

## Andy Grove 时间线

| 时间 | 事件 | 意义 |
|------|------|------|
| 1936-09-02 | 出生于布达佩斯，原名 András Gróf | 战争与动荡背景塑造了他的现实主义 |
| 1956-1957 | 逃离匈牙利，移民美国 | 强烈的生存视角与迁移经历 |
| 1963 | 在 UC Berkeley 获得化学工程博士 | 工程与分析训练底座 |
| 1963-1967 | 在 Fairchild Semiconductor 工作 | 接近半导体核心圈层 |
| 1968 | 作为 Intel 最早成员加入公司 | 从创业早期就深度参与系统建设 |
| 1979 | 成为 Intel 总裁 | 从技术执行走向全面经营 |
| 1983 | 出版 `High Output Management` | 系统化输出其管理框架 |
| 1987 | 成为 Intel CEO | 进入最高经营层 |
| 1990s | 主导 Intel 在 PC 时代的核心扩张 | 组织和战略能力进入成熟期 |
| 1996/1999 | `Only the Paranoid Survive` 面世 | 把战略拐点方法论显性化 |
| 1997 | 被《时代》评为年度人物 | 影响力从企业内部扩散到社会层面 |
| 1998 | 卸任 CEO，继续担任董事长 | 完成权力交接 |
| 2005 | 结束 Intel 董事长任期 | 离开一线经营角色 |
| 2016-03-21 | 去世 | 其管理方法仍持续影响科技行业 |

## 时间线里的结构性变化

### 第一阶段：工程师与运营者

这个阶段的 Grove 更像系统设计师。
重点是把复杂技术组织做成能稳定输出的机器。

### 第二阶段：CEO 与战略校准者

这个阶段的 Grove 更关注：

- 哪些信号预示行业正在变
- 公司什么时候该承认旧规则失效
- 如何让组织在混乱中重建秩序

### 第三阶段：方法论输出者

离开一线后，Grove 的影响力更多体现在：

- 书
- 课程
- 组织方法的扩散
- 对健康和公共问题的系统化介入

## 对 skill 使用的意义

时间线说明了 Andy Grove 适合被用在三类问题上：

1. 扩张中的执行系统
2. 指标与控制回路
3. 战略拐点识别与收口

不适合把他当成纯消费者产品灵感源。
